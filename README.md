# TorchDelta

## Concept

`torchdelta` simplifies using `DeltaLake` tables for training using PyTorch. 

## Requirements

- Python Version \> 3.8
- `pip` or `conda`

## Installation

- with `pip`:

```
pip install git+https://github.com/mshtelma/torchdelta
```
